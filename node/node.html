Quick Start here¶

Following order of the resources below is a preferred approach to learn progressively and quickly.
You will learn to write a full fledged node.js web app that uses express, passport and mongoose (MongoDB)
Additional sections are optional, but they will help in understanding the core concepts much clearer

  • 1. What is Node.js? - Know about Node.js, a high level understanding
  • [Additional] Introduction to Node.js with Ryan Dahl - Know about Node.js from the creator of Node
  • [Additional] Node.js: JavaScript on the Server - Node.js purpose and evolution
  • 2. A Simple App - Start creating a simple node web server app
  • [Additional] Understanding Web Application Stack & Fundamentals
  • [Additional] Absolute Beginners Guide to Node.js - Yet another, but has file I/O details
  • 3. Beginning routing in Node.js - Use Express.js to do more in your web app
  • 4. Cleaning Up With Express Routing - Get involved in Express.js
  • 5. Middleware in Express - Find out what makes Express.js powerful & flexible
  • 6. Deep into custom middleware - Understanding middleware in Express.js
  • 7. Node code in other files - Introducing modules, exports in Node.js
  • [Additional] Modules in Node
  • [Additional] Third-party Modules and NPM
  • 8. How to structure a Node app - Find a way to organize your project
    • There are many approach to structure your project. More details will be added in advanced/other resources.
  • 9. View templates in Node.js with Jade - Organize/Abstract your view with templates
    • Templates is a basic method used in Express.js to bind data with view. There are many template and view engines available in Express through consolidate middleware. Jade is simple and elegant.
  • 10. More advanced Jade
  • 11. MongoDB + Node = Mongoose - Add database to your app
  • [Optional] MongoDB plugin - Learn to use mongodb package to connect to MongoDB server
  • 12. Unique records in Mongoose
  • 13. Custom validations in Mongoose
  • Refer here to continue the series
  • TODO - Cover the following
    • Core Modules - Process, OS, File System, Buffer, Events, Cluster, ...
    • Error Handling
    • Others - authentication, authorization, web sockets, ...
